# Нефункциональные тест кейсы

## Проверка производительности

### Тест кейс "Обработка 10 000 строк"

Цель:
Проверить, что скрипт выполняется не больше 1 секунды.

given
- генерация CSV файла размером 10000 строк
- каждая строка содержит:
    - user
    - корректную дату
    - категорию
    - сумму

when
- запуск calculate_averages.sh

then
- exit code == 0
- время выполнения <= 1 секунды
- создан файл averages.csv


---
